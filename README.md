## What I Changed & Why

| Change | Why |
|--------|-----|
| **Removed "Hey, I'm" casual opener** | Elite developers don't introduce themselves like a Discord message. Name alone carries weight. |
| **"Software Engineer & Creative Technologist"** | "Full Stack Web Developer" is accurate but generic. This title communicates depth + creativity. |
| **"Arusha, Tanzania → Shipping globally"** | Location as a feature, not a footnote. Meridian direction in one line. |
| **Right-aligned GIF** | Visual balance. GIF adds personality without dominating. |
| **Removed emoji-heavy section headers** | The rocket, brain, lightning bolts read as junior. Clean typography reads as senior. |
| **Renamed "About Me" to integrated opening** | The intro paragraph does the work without labeling itself. |
| **Renamed "Tech Stack" to "Craft"** | "Stack" is what you use. "Craft" is what you do with it. |
| **Removed badge icons from tech list** | Badge walls scream "bootcamp graduate." A clean code block is more confident. |
| **Added "What I Build" as a table** | Concrete deliverables, not abstract skills. A hiring manager sees exactly what you produce. |
| **"Currently Exploring" replaces "Learning"** | "Learning" implies you don't know it yet. "Exploring" implies active curiosity from a position of strength. |
| **Custom badge colors (#C4622D)** | Meridian terracotta accent. Consistent with your new portfolio direction. No default badge colors. |
| **Kept the closing quote** | It's genuinely good. It earned its place. |
| **Removed `shadcn/ui` from craft block** | It's a UI library, not a skill. Keep the block tight to things that demonstrate depth. |
| **Removed "mindset" from the code block** | Philosophy is a separate statement now — cleaner separation. |
